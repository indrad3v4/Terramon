## 📣 Pitch Deck для openvc.app

### Slide 1: Problem
```
AI is trapped in chatbots.
Users type → AI talks back. That's it.
No world. No stakes. No magic.

People want to PLAY with AI, not chat with it.
```

### Slide 2: Solution — Terramon
```
Thought → IntentRouter → Codex Creature → World Event

You don't "talk to AI".
You SUMMON a creature from the nature of your thought.
```

### Slide 3: How It Works
```
1. User types any thought ("I feel lost tonight")
2. IntentRouter classifies: wisdom → Sage summoned
3. Sage appears in the world map — with memory, territory, lore
4. Rare summons cost Lightning sats (Bitcoin micropayments)
5. Thoughts are signed on-chain — immutable quest log
```

### Slide 4: Why Now
```
- LLMs are cheap enough to classify ANY thought in real-time
- Lightning Network makes microtransactions feasible
- AMD GPUs + Fireworks AI = open-source inference at scale
- Mobile-first Reflex UI runs everywhere
```

### Slide 5: Market
```
- AI gaming market: $10B+ by 2027 (McKinsey)
- "AI companion" apps growing 300% YoY (data.ai)
- Web3 gaming: $50B+ projected (Bitcoin Lightning = zero gas fees)
- Niche: AI thought-to-creature — NO competitors in this exact slot
```

### Slide 6: Business Model
```
Tier        | Price     | What
FREE        | $0        | 5 summons/day, basic creatures
SUMMONER    | $9.99/mo  | 50 summons/day, priority queue
RARE        | 1000 sats | Rare codex creatures (Lightning)
BUILDER     | $4,999    | Custom agent system (architect service)
```

### Slide 7: Tech Stack
```
Frontend   → Reflex (Python, mobile-first, no JS)
Backend    → Hexagonal + Event-Driven (Python)
AI         → Fireworks AI on AMD GPU (classifier + creature responses)
Memory     → SQLite → PostgreSQL (via hexagonal ports)
Payments   → Bitcoin Lightning Network
Deploy     → Docker + AMD Developer Cloud
Identity   → Decentralized DID (wallet keys)
Lenses     → Jesse Schell's 100 Game Design Lenses (applied every sprint)
```
