## СТРАТЭГІЯ 1: "AI Mentor" — Твой розум як геймпад

| Элемент | Значэнне |
|---|---|
| **Праблема** | Людзі не ведаюць, як выкарыстоўваць AI для сябе. Чат-боты — гэта нудна. Трэба *гуляць*, каб вучыцца. |
| **Рашэнне** | Terramon — першая AI-native гульня. Ты вучышся промпт-інжынірынгу праз гульню: лепш думаеш → лепшыя стварэнні. |
| **Нерв** | "Я хачу разумець AI, але не хачу чытаць дакументацыю" + "Але трэба нешта рабіць, каб не адстаць" = **Я вучуся AI праз гульню, сам таго не заўважаючы** |
| **Інсайт** | People learn AI best when they don't know they're learning. |
| **Хто плаціць** | **B2B:** кампаніі, якія хочуць навучыць супрацоўнікаў AI ($49/seat/mo) + **B2C:** AI-энтузіясты ($9.99/mo) |
| **Market size** | AI literacy training: $8B by 2027 (Gartner) |
| **Прыклад победы да 34** | 500 B2B seats, €24k/mo MRR |

## СТРАТЭГІЯ 4: "Therapy Companion" — Тэрапія праз гульню

| Элемент | Значэнне |
|---|---|
| **Праблема** | Псіхатэрапія дорага (€80-150/сесія). Journaling apps нудныя. Людзі хочуць *працаваць* з думкамі, але без цяжару тэрапіі. |
| **Рашэнне** | Terramon класіфікуе твае думкі і прызывае стварэнне, якое адпавядае твайму настрою. "I feel lost" → Sage. "I'm angry" → Strategist. Гэта як journaling + AI coach + game. |
| **Нерв** | "Я не магу дазволіць сабе тэрапію, але мае думкі ціснуць" + "Веданне дзённіка не дапамагае" = **Я гуляю ў гульню, і мае думкі становяцца зразумелымі, без тэрапеўта** |
| **Інсайт** | Gaming is the cheapest therapy. Summoning is the new journaling. |
| **Хто плаціць** | **B2C:** людзі 25-40 з трывогай, self-care ($9.99/mo) + **B2B:** корпаратыўны well-being ($499/mo для каманды) |
| **Market size** | Digital therapy: $10B by 2027 |
| **Прыклад победы да 34** | 1000 active users, €8k/mo MRR |
